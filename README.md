# HelpDesk M-Autotech

Sistema de gestión de tickets para soporte TI en empresa de manufactura automotriz.

## Tecnologías

- Backend: Node.js
- Frontend: React
- Base de datos: PostgreSQL

## Funcionalidades

- Gestión de tickets
- Asignación de técnicos
- Dashboard de métricas
- Chat interno
